# AGENTS.md

このドキュメントは、AI開発エージェントがこのプロジェクトで作業する際のガイドラインを提供します。

## プロジェクト概要

このプロジェクトは、仮想通貨取引所間の裁定取引機会を特定し、関連するデータ分析を行うためのアプリケーションです。コマンドラインツールとしてのコア機能と、Streamlitを利用したWeb UIの両方を提供します。

## プロジェクト構成

プロジェクトは、以下の2つの主要なコンポーネントに分かれています。

-   `/core`: 裁定取引の計算、データ取得、設定管理など、アプリケーションの中核となるビジネスロジックが含まれています。このディレクトリは自己完結しており、UIコンポーネントから独立して実行可能です。
-   `/streamlit_app`: Streamlitを使用して構築されたWeb UIに関連するすべてのファイルが含まれています。データの可視化やユーザーとのインタラクションを担当します。

## 設定管理

アプリケーションの設定は、プロジェクトのルートディレクトリにある`config.ini`ファイルで一元管理されています。APIキーや手数料率、監視対象の通貨ペアなどのパラメータは、コードを直接変更することなく、このファイルで設定・変更してください。

## 依存関係の管理

依存関係は、各コンポーネントごとに独立して管理されています。

-   `/core/requirements.txt`: コアロジックに必要なPythonライブラリが記載されています。
-   `/streamlit_app/requirements.txt`: Streamlitアプリケーションに必要なライブラリが記載されています。

新しいライブラリを追加する際は、適切な`requirements.txt`ファイルに追記してください。

## 開発ルール

-   **開発言語:** このプロジェクトの主要な開発言語は **Python** です。
-   **UIフレームワーク:** Web UIの開発には **Streamlit** を第一候補とします。
-   **関心事の分離:** `core`と`streamlit_app`の責務分離を維持してください。
    -   普遍的なビジネスロジックや計算処理は`/core`ディレクトリに追加します。
    -   UIの表示やユーザーインタフェースに関連するコードは`/streamlit_app`ディレクトリに配置します。
-   **コードの再利用:** `streamlit_app`から`core`の機能を呼び出すことは推奨されますが、その逆（`core`から`streamlit_app`の機能を呼び出すこと）は避けてください。
