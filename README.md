# 暗号資産裁定取引・分析ツール

## 1. プロジェクトの目的

このプロジェクトは、暗号資産（仮想通貨）取引所間の裁定取引（アービトラージ）機会の発見と、関連するデータ分析を行うためのツールです。裁定取引を計算するコアなコマンドラインツールと、データ可視化のためのStreamlit製Webアプリケーションの2つの主要コンポーネントで構成されています。

## 2. 主な特徴

-   **裁定取引の検出:** GMOコインとbitbank間の潜在的な裁定取引機会をリアルタイムで特定します。
-   **Webインターフェース:** Streamlitを利用したユーザーフレンドリーなWeb UIを提供し、対話的なデータ分析や可視化を実現します。
-   **モジュール化された構造:** コアロジックとWebアプリケーションを分離し、保守性と拡張性を高めています。
-   **外部設定ファイル:** `config.ini`ファイルを通じて、取引所の手数料率やAPIエンドポイントなどを容易に設定できます。

## 3. セットアップ方法

1.  **プロジェクトのルートディレクトリに移動します。**
2.  **依存関係をインストールします:**
    ```bash
    pip install -r requirements.txt
    ```
    *注意: 依存ライブラリが多いため、インストールには時間がかかる場合があります。*

## 4. 使い方

### 4.1. コア裁定取引ツールの実行

裁定取引の機会を検出するには、プロジェクトのルートディレクトリから以下のコマンドを実行します。

```bash
python -m core.run_arbitrage
```
ツールが設定された取引所から最新の板情報を取得し、裁定取引の機会が見つかれば結果を出力します。

### 4.2. Streamlit Webアプリケーションの起動

1.  **プロジェクトのルートディレクトリにいることを確認します。**
2.  **以下のコマンドでStreamlitアプリを起動します:**
    ```bash
    streamlit run index.py
    ```
3.  WebブラウザでStreamlitから提供されたローカルURL（通常は `http://localhost:8501`）を開きます。
